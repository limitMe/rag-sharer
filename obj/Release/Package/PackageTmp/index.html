大标题

作为一个非常激进的童鞋，我一下子就选择了windows server 2012 x64的操作系统，安装了iis，不知道现在能不能正确使用webAPI访问了。当然选择2012版，其自带的.net 4.5更给我提供了现在所有能够用到资源的应用环境。
